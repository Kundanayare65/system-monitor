# System Monitor

A real-time system monitoring dashboard built with Rust and Axum.

## 🚀 Live Demo

https://system-monitor-ase8.onrender.com

## Features

- Real-time CPU usage
- Memory monitoring
- Disk usage
- Network statistics
- Process information
- Light/Dark theme

## Tech Stack

- Rust
- Axum
- Tokio
- Sysinfo

## Run Locally

```bash
cargo run
